# bushfire-damage-claims-analysis
This project analyses bushfire damage claim data using statistical and machine learning techniques implemented in R. The main objective of the analysis is to identify the key factors that influence the monetary value of property damage claims caused by bushfires. The dataset contains information about fire intensity, distance from the fire source, building characteristics, weather conditions, insurance coverage, and other environmental variables. By analysing these factors, the project aims to understand how different conditions contribute to the level of damage experienced by properties during bushfire events.

The analysis begins with data preprocessing and exploratory data analysis to understand the structure and distribution of the dataset. Techniques such as histograms, box plots, scatter plots, and correlation analysis are used to identify patterns and relationships between variables. Multiple linear regression is then applied to evaluate how different predictors influence damage claims and to determine the most significant variables affecting the outcome. Model improvement techniques, including interaction terms and polynomial terms, are also explored to refine the regression model and improve its explanatory power.

In addition to statistical modelling, machine learning techniques are used to classify whether property damage claims exceed a specific threshold. Support Vector Machine models with linear, polynomial, and radial kernels are implemented and tuned using cross-validation to identify the best performing model. The results highlight the most important predictors influencing damage claims and demonstrate how data analysis and predictive modelling can support decision-making in insurance risk assessment. This project was developed as part of coursework at Western Sydney University and demonstrates practical skills in data analysis, statistical modelling, and machine learning using R.
